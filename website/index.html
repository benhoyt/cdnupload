cdnupload

"Caching is Better with Hashing." Keep your website fast, and don't break CSS and JavaScript when you deploy.

cdnupload uploads your website's static files to a CDN with a content-based hash in the filename, giving great caching while avoiding versioning issues.

  • Fast and simple to integrate
  • Uses web performance best practices
  • Works with web apps written in any language
  • Written in Python (runs on Python 2 and 3)

Trivial to install:

$ pip install cdnupload

Easy to integrate (Python, but any language works):

import cdnupload, settings

def static_url(rel_path):
    return '//mycdn.com' + settings.statics[rel_path]

def init_server():
    source = cdnupload.FileSource('/site/static')
    settings.statics = source.build_key_map()

Simple to use:

$ cdnupload /site/static s3://static-bucket \
            --key-map=static.json
uploading script.js to script_8f3283c6342816f7.js
uploading style.css to style_deadbeef01234567.css
$ deploy-my-app
