cdnupload

No more broken CSS and JavaScript after you deploy.

cdnupload avoids versioning issues with static files: it uploads and manages your static files with a content-based hash in the filename, to enable you to easily implement caching best-practices

A Python-based tool to upload static files from a source directory to a destination directory or Amazon S3 bucket, with content-based hash in filenames for versioning.
